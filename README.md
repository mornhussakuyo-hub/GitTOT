
# GitTOT

一个简单的 Python 小工具，用来统计 Git 仓库中各个小时的代码增加/删除行数，并在终端输出分布图。

## 功能

- 读取当前 Git 仓库的提交记录
- 统计每个小时的代码增加行数与删除行数
- 在终端显示左右分布图
  - 左侧表示删除
  - 右侧表示增加

## 使用方法

### 1. 进入 Git 仓库目录

```bash
cd 你的项目目录
````

### 2. 运行脚本

```bash
python gittot.py
```

## 输出说明

程序会按 `00:00` 到 `23:00` 统计代码改动情况。

示意：

* 左边条形：删除行数 `-`
* 中间竖线：0 轴
* 右边条形：增加行数 `+`

同时会显示总增加行数和总删除行数。

## 运行要求

* Python 3
* Git
* 当前目录必须是一个 Git 仓库
